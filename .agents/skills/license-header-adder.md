# Agregador de Encabezados NANDU LSD

Esta habilidad garantiza que todos los archivos nuevos creados para este proyecto científico tengan la atribución correcta a la Universidad de Buenos Aires y a los autores.

## Instrucciones Estrictas para la IA:
1. **Lee la plantilla:** Siempre lee el contenido de `resources/HEADER.txt`.
2. **Aplicación Obligatoria:** Cada vez que el usuario te pida crear un archivo nuevo (ej. `.py`, `.json`, `.md`), DEBES poner este bloque de texto en la línea 1.
3. **Autocompletado:** Reemplaza la parte que dice "[La IA debe completar una breve descripción del script aquí]" por un resumen real de lo que hace el código que acabas de escribir.
4. **Sintaxis:** Como este es un proyecto en Python, mantén los símbolos `#` para que el código no dé error de sintaxis.

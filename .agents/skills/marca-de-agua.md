# Agregador de Marca de Agua y Créditos

Esta habilidad garantiza que todos los archivos nuevos creados para este proyecto contengan la atribución correcta del proyecto y del autor principal.

## Instrucciones Estrictas para la IA:
1. **Aplicación Obligatoria:** Cada vez que el usuario te pida crear un archivo nuevo (ej. `.py`, `.js`, `.cpp`), DEBES poner el siguiente bloque de texto en la parte superior del archivo.
2. **Plantilla de Marca de Agua:**
```python
# =====================================================================
# Proyecto: Generador de videos musicales de Rebeldia Cósmica
# Autor: Santiago Prado (Estudiante de Física y Compositor)
# Descripción: [La IA debe completar una breve descripción del script aquí]
# =====================================================================
```
3. **Autocompletado:** Reemplaza la parte de "Descripción" por un resumen real de lo que hace el código.
4. **Sintaxis:** Ajusta el formato de comentario según el lenguaje de programación (ej. `#` para Python, `//` para JavaScript o C++).
